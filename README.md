# v1.0.0 — Initial Release

Primeira versão do meu site pessoal (portfolio).

## Sobre

Site desenvolvido para centralizar projetos, experiência e informações profissionais em um único lugar.

## Funcionalidades

* 5 páginas (home, projetos, cv, blog, contato)
* Layout responsivo (mobile e desktop)
* Otimizado para SEO
* Otimizado para google lighthouse

## Stack

* Frontend: HTML/CSS/JS vanilla
* Deploy: Cloudflare Domain + Github pages

Versão inicial estável.
